# Release 2026.02.5-RELEASE

## Highlights

- **Phase 12 Complete**: Tax Data Management — tax detail entry UI/API, client management, fiscal periods, PPN formula fix, auto-populate from templates, enhanced tax reports, retrofitted 2025 data, tax export API.
- **Phase 13 Complete**: OpenAPI Migration — replaced hand-maintained `capabilities.json` with auto-generated OpenAPI spec via springdoc-openapi 3.0.1.

## What's New

### Phase 12: Tax Data Management

- **12.1** PPN formula fix: `amount × 11/111` → `DPP = amount × 11/12, PPN = DPP × 12%` (DPP Nilai Lain per PMK 131/2024)
- **12.2** Tax detail entry UI on transaction detail page (faktur pajak, bupot, counterparty fields)
- **12.3** Tax detail & document REST API (6 tax detail endpoints + 4 document endpoints)
- **12.4** Auto-populate tax details when posting via PPN/PPh templates
- **12.5** Client management UI with NPWP validation, CSV import, linked projects/invoices
- **12.6** Fiscal period management (open/close periods, block posting in closed periods)
- **12.7** Tax report enhancements (PPN/PPh23 detail, cross-check, rekonsiliasi fiskal, PPh Badan calculator)
- **12.8** Tax deadline updates per PMK 81/2024 (payment deadlines 10th → 15th)
- **12.9** Retrofitted 2025 data with correct PPN formula and tax details
- **12.10** Tax export API (7 endpoints: e-Faktur, bupot, PPN/PPh detail, rekonsiliasi fiskal, PPh Badan)
- **12.11** PPN documentation update (DPP Nilai Lain explanation in user manual)

### Phase 13: OpenAPI Migration

- springdoc-openapi 3.0.1 with Swagger UI at `/swagger-ui.html`
- Auto-generated OpenAPI 3.1.0 spec at `/v3/api-docs` (67 API paths, 145 schemas)
- `@Tag` on all 11 API controllers, `@Hidden` on 9 MVC controllers
- 5 x-extensions: `x-authentication`, `x-workflows` (13), `x-csv-files` (16), `x-industries` (4), `x-error-codes` (7)
- 11 functional tests (OpenApiTest)
