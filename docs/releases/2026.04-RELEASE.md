## Highlights

First-run setup wizard makes Balaka usable from a fresh database without manual seed loading. Docker image is now published to Docker Hub and GHCR. Free-form journal entry UI lets accountants post arbitrary debit/credit lines without templates. Chart of Accounts and transaction-purge APIs round out the REST surface. Documentation has been restructured into a multi-page site for five distinct audiences with full per-industry tutorials.

## What's New

### First-Run Setup Wizard
- `GET /setup` form creates the first admin user and loads a selected industry seed pack (it-service, online-seller, coffee-shop, campus)
- Interceptor redirects all browser requests to `/setup` while the users table is empty; no-op once any user exists
- Industry seed packs bundled into the JAR via Maven resources and copied into the Docker image
- 4 Playwright functional tests verify form submission, BCrypt password hashing, ADMIN role assignment, and seed import
