## Klien dan Proyek

| Klien | Tipe | Proyek | Template Pendapatan |
|-------|------|--------|-------------------|
| Bank Mandiri | BUMN | Core Banking Modernization | Pendapatan + PPN + PPh 23 |
| Telkom | BUMN | Network Monitoring System | Pendapatan + PPN + PPh 23 |
| PLN | BUMN | Smart Grid Analytics | Pendapatan BUMN (FP 03) |
